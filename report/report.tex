\section*{Описание}
\addcontentsline{toc}{section}{Описание}

Приложение представляет из себя консольную утилиту, которая в виде аргументов командной строки получает входные данные и выводит результат на экран. 

При запуске приложения с ключом -h (-help, --help) на экран будет выведена информация о том, как пользоваться данной утилитой. Программа ожидает 3 аргумента - исходное число; CC, в которой оно представлено; СС, в которой его необходимо представить. Выводом будет результат работы.
